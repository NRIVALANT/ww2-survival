▸ Comment jouer

  1. Télécharge et lance ww2survival.exe — si Windows bloque, clique Informations complémentaires → Exécuter quand même
  2. Choisis Multijoueur → Rejoindre
  3. Entre l'IP du serveur : 16.171.254.223
  4. Le 1er joueur connecté est l'hôte — il lance la partie
  5. Survivez ensemble le plus longtemps possible !
